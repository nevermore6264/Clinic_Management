# DANH MỤC CHỮ VIẾT TẮT TIẾNG ANH

| STT | Viết tắt | Thuật ngữ đầy đủ (tiếng Anh) | Nghĩa tiếng Việt |
|:---:|:---------|:----------------------------|:-----------------|
| 1 | API | Application Programming Interface | Giao diện lập trình ứng dụng |
| 2 | CRUD | Create, Read, Update, Delete | Tạo, đọc, cập nhật, xóa dữ liệu |
| 3 | CSS | Cascading Style Sheets | Ngôn ngữ định dạng tầng kiểu |
| 4 | DTO | Data Transfer Object | Đối tượng truyền dữ liệu |
| 5 | ERD | Entity–Relationship Diagram | Sơ đồ thực thể – quan hệ |
| 6 | HTML | HyperText Markup Language | Ngôn ngữ đánh dấu siêu văn bản |
| 7 | HTTP | HyperText Transfer Protocol | Giao thức truyền tải siêu văn bản |
| 8 | HTTPS | HTTP Secure | HTTP bảo mật (mã hóa TLS) |
| 9 | JSON | JavaScript Object Notation | Định dạng trao đổi dữ liệu dạng văn bản |
| 10 | JPA | Java Persistence API | Giao diện lưu trữ bền vững trong Java |
| 11 | JWT | JSON Web Token | Chuẩn token dùng cho xác thực |
| 12 | ORM | Object–Relational Mapping | Ánh xạ đối tượng – quan hệ |
| 13 | REST | Representational State Transfer | Kiến trúc giao tiếp qua tài nguyên và HTTP |
| 14 | SQL | Structured Query Language | Ngôn ngữ truy vấn có cấu trúc |
| 15 | STOMP | Simple Text Oriented Messaging Protocol | Giao thức nhắn tin đơn giản (thường dùng với WebSocket) |
| 16 | TLS | Transport Layer Security | Bảo mật tầng truyền tải |
| 17 | UI | User Interface | Giao diện người dùng |
| 18 | UML | Unified Modeling Language | Ngôn ngữ mô hình hóa thống nhất |
| 19 | URL | Uniform Resource Locator | Định vị tài nguyên thống nhất |
| 20 | WebSocket | WebSocket Protocol | Giao thức kết nối hai chiều, thời gian thực |

**Ghi chú:**  
- Tên riêng sản phẩm/công cụ (**MySQL**, **Spring Boot**, **Next.js**, **React**, **TypeScript**) thường không đưa vào bảng “viết tắt” mà ghi trong phần công nghệ; nếu trường yêu cầu, có thể thêm dòng riêng (không có dạng viết tắt chuẩn).  
- Chỉ giữ các dòng tương ứng với thuật ngữ thực sự dùng trong báo cáo; đánh lại STT cho đúng.
