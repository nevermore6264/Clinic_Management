# DANH MỤC CHỮ VIẾT TẮT

| STT | Chữ viết tắt | Nghĩa đầy đủ |
|:---:|:-------------|:-------------|
| 1 | API | Giao diện lập trình ứng dụng (*Application Programming Interface*) |
| 2 | CSDL | Cơ sở dữ liệu |
| 3 | CRUD | Tạo – Đọc – Cập nhật – Xóa (*Create – Read – Update – Delete*) |
| 4 | CSS | Ngôn ngữ định dạng tầng kiểu (*Cascading Style Sheets*) |
| 5 | DTO | Đối tượng truyền dữ liệu (*Data Transfer Object*) |
| 6 | ERD | Sơ đồ thực thể – quan hệ (*Entity–Relationship Diagram*) |
| 7 | HTML | Ngôn ngữ đánh dấu siêu văn bản (*HyperText Markup Language*) |
| 8 | HTTP/HTTPS | Giao thức truyền tải siêu văn bản (phiên bản mã hóa *HTTPS*) |
| 9 | JSON | Định dạng dữ liệu *JavaScript Object Notation* |
| 10 | JPA | Giao diện lưu trữ Java (*Java Persistence API*) |
| 11 | JWT | Token Web JSON (*JSON Web Token*) |
| 12 | MySQL | Hệ quản trị cơ sở dữ liệu quan hệ *MySQL* |
| 13 | Next.js | Framework *Next.js* (React) |
| 14 | ORM | Ánh xạ đối tượng – quan hệ (*Object–Relational Mapping*) |
| 15 | REST | Kiến trúc chuyển trạng thái đại diện (*Representational State Transfer*) |
| 16 | SQL | Ngôn ngữ truy vấn có cấu trúc (*Structured Query Language*) |
| 17 | STOMP | Giao thức nhắn tin đơn giản (*Simple Text Oriented Messaging Protocol*) |
| 18 | TS | TypeScript |
| 19 | UI | Giao diện người dùng (*User Interface*) |
| 20 | UML | Ngôn ngữ mô hình hóa thống nhất (*Unified Modeling Language*) |
| 21 | URL | Định vị tài nguyên thống nhất (*Uniform Resource Locator*) |
| 22 | WebSocket | Giao thức *WebSocket* (kết nối hai chiều, thời gian thực) |

**Ghi chú:** Chỉ giữ các dòng tương ứng với từ viết tắt thực sự xuất hiện trong báo cáo; có thể thêm/bớt dòng và đánh lại STT cho đúng.
