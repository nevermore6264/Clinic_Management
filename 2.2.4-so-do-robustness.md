# 2.2.4. Sơ đồ Robustness (Boundary – Control – Entity)

_Sơ đồ Robustness (Jacobson) mô tả tương tác giữa **Tác nhân**, **Boundary** (giao diện / ranh giới hệ thống), **Control** (điều khiển nghiệp vụ) và **Entity** (dữ liệu / bảng CSDL). Mỗi use case có một file PlantUML riêng trong thư mục `diagrams/robustness-uc/`._

**Cách xuất hình:** mở file `.puml` trong PlantUML → Export PNG/SVG → chèn vào Word (ví dụ **Hình 2.22** cho UC Đăng nhập).

**Sinh lại toàn bộ file (sau khi chỉnh script):** `python scripts/gen_robustness_uc.py`
